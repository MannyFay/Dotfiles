# Raycast
| Function                  | Command       |
|:--------------------------|:-------------:|
| Open Raycast:             | `alt`+`space` |
| Back to previous page:    | `esc`         |
| Back to main search page: | `cmd`+`esc`   |
| Close Raycast window:     | `cmd`+ `w`    |
| Open Raycast preferences: | `cmd`+`,`     |
| Navigate down:            | `ctrl`+`j`    |
| Navigate up:              | `ctrl`+`k`    |
| Navigate right:           | `ctrl`+`l`    |
| Navigate left:            | `ctrl`+`h`    |
